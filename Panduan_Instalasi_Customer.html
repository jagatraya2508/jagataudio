Panduan Instalasi JagatAudio

Terima kasih telah menggunakan JagatAudio! Aplikasi cerdas ini dilindungi oleh sistem lisensi tingkat tinggi yang terikat secara khusus pada perangkat keras (Hardware ID) komputer Anda.

Langkah 1: Persiapan File

Pastikan Anda telah menerima file berikut ini dari tim Admin:
  1. Folder / File Aplikasi JagatAudio (biasanya dikirim dalam bentuk .zip atau folder utuh).
  2. File Lisensi Aktif berekstensi .lic (contoh: JagatAudio_NamaAnda.lic).

Langkah 2: Menjalankan Aplikasi

  1. Jika Anda menerima file .zip, extract (keluarkan) isinya terlebih dahulu.
  2. Pindahkan folder JagatAudio ke tempat yang aman di komputer Anda (misalnya di D:\ atau Documents).
  3. Buka folder JagatAudio tersebut.
  4. Cari dan klik dua kali (Double Click) pada file bernama JagatAudio.exe.
  5. Tunggu sekitar 2-3 detik. Aplikasi akan secara otomatis membuka browser menuju halaman utama.

Langkah 3: Mengaktifkan Lisensi

Saat aplikasi pertama kali dibuka, layar akan terkunci pada halaman Aktivasi Lisensi Diperlukan.

  1. Pada halaman tersebut, temukan area kotak putus-putus bertuliskan Upload File Lisensi (.lic).
  2. Tarik dan lepas (Drag & Drop) file .lic milik Anda ke dalam kotak tersebut.
  3. Alternatifnya, Anda dapat mengklik kotak tersebut lalu mencari dan memilih file .lic Anda secara manual.
  4. Jika lisensi valid, halaman akan otomatis memuat ulang, dan Anda siap menggunakan fitur-fitur canggih JagatAudio!
Catatan Penting: File lisensi (.lic) milik Anda bersifat unik dan hanya berlaku pada komputer ini. File tersebut tidak dapat dicopy, dipindahkan, atau digunakan di komputer lain.

Butuh Bantuan?

Jika Anda mengalami kendala saat masuk (misalnya muncul pesan Hardware ID tidak valid), pastikan Anda memberikan kode Hardware ID (tulisan berwarna biru di layar aplikasi Anda) kepada Admin untuk membuat ulang file .lic yang tepat.
